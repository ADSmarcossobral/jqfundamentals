events

Files: sandbox.html, sandbox.js

summary exercise

Files: index.html, slideshow.js

  • move ul#hero to the top of the page, before the h1
  • create a slideshow from the list items in ul#hero with the following behavior:
    • first image fades up on document ready

    • add the ability to navigate through the slideshow

ajax

get specials details using ajax

Files: index.html, specials.js

  • append a div after the form in div#specials; this will be where you put information about the special once you receive it
  • remove the submit button from the form
  • make it so that changing the select menu sends an ajax request
    • the request should point to json/specials.json (you may want to look at the file to see the data you receive)
    • the request should expect to receive json back from the server
    • use the json response you receive to add some html elements to the container div you appended

load menu item details using ajax

Files: menu.html, menus.js

  • there is a list item for each type of meal (lunch, dinner, brunch); create a container div at the end of each of these list items. this is where you'll put the html response to your ajax request
  • bind an event to #menus a
    • this event will need to get the "hash" portion of the href attribute of the a that was clicked. hint: 
      var myUrl = $(this).attr('href'); var hash = myUrl.split('#')[1];
    • use the hash to build a url for the ajax request: html/stirfry.html, for example
    • send an ajax request to the url
    • use the response of the request to populate the container div you created
    • cache the request so you don't have to get the data again
