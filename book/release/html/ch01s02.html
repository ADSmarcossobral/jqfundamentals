Software

Você precisará das seguintes ferramentas para aproveitar as aulas ao máximo:

  • O navegador Firefox

  • A extensão Firebug para o Firefox

  • Um editor de textos

  • For the Ajax portions: A local server (such as MAMP or WAMP), or an FTP or SSH client to access a remote server.

  • Para as partes com Ajax: um servidor local (como o MAMP ou WAMP), ou um cliente FTP ou SSH para acessar um servidor remoto.
